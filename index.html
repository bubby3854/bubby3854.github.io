소개

안녕하세요. 저는 추창민입니다. 신입 웹 개발자로, 웹 애플리케이션 개발에 열정을 가지고 있습니다. 새로운 기술을 배우고 적용하는 것을 좋아하며, 팀 프로젝트에서 협업을 통해 최상의 결과를 만들어내는 것에 자부심을 느낍니다.

기술 스택

  • Java
  • JavaScript
  • Spring Boot
  • MariaDB
  • Git
  • Flutter

프로젝트 1

구인구직 플랫폼

설명: 사용자 친화적인 인터페이스, 교육내용을 적용하여 개발

사용 기술:Spring Boot-mybatis, MariaDB

역할: 구인구직 사이트의 전반적인 로그인기능과 관리자모드에서의 관리자기능을 개발

주요 성과: 팀원들과 협엽하여 Git을 사용하여기본적인 CRUD 사용과 데이터베이스 연동으로 웹 어플리케이션을 개발

링크: GitHub 저장소

프로젝트 2

구인구직 플랫폼

설명: Rest Api 활용하여 설계 및 구현으로 웹 어플리케이션 개발

사용 기술:Spring Boot-JPA, JavaScript, MariaDB

역할:JPA를 사용하여 관리자 대시보드 및 추천 공고및 AI 맞춤추천 기능을 개발.

주요 성과:git을 사용하여 팀 단위 프로젝트를 JPA 및 Rest Api를 사용하고 데이터베이스 연동 으로 웹 어플리케이션을 개발

링크: GitHub 저장소

프로젝트 3

부산 관광홍보 사이트 개발

설명: Rest Api 활용과 공공데이터 활용하여 홈페이지 구성하였으며, 여러가지 API를 사용하여 개발

사용 기술:Spring Boot-JPA, MariaDB, JavaScript

역할:GPT api, 날씨 Api, 구글맵 Api, 공공데이터 Api를 사용하여 필요한 정보를 DB에 저장해서 활용

주요 성과:Api 및 RestConroller 사용하여 웹 어플리케이션을 개발

링크: GitHub 저장소
